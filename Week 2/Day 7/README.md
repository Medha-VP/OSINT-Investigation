# Advanced Google Dorking – IBM

## Overview

This repository contains an Open-Source Intelligence (OSINT) investigation of **IBM** using advanced Google Dorking techniques. The investigation demonstrates how advanced Google search operators can be used to discover publicly indexed resources while following ethical and passive information-gathering practices.

## Objective

- Perform passive OSINT on IBM's public web presence.
- Use advanced Google Dorks to locate publicly indexed resources.
- Discover technical documents, reports, presentations, and career pages.
- Understand how search engines index organizational content.
- Practice ethical and responsible OSINT techniques.

## Google Search Operators Used

The investigation includes the use of advanced search operators such as:

- `site:`
- `filetype:`
- `intitle:`
- `inurl:`
- `intext:`
- `cache:`
- `related:`
- `AROUND()`
- `OR`
- `-` (Exclude Operator)

These operators help narrow search results and locate publicly available information more efficiently.

## Investigation Methodology

The investigation was conducted using publicly available information indexed by Google. Advanced Google Dorks were applied to identify:

- Annual reports
- AI research papers
- Technical documentation
- Product brochures
- Career pages
- Public presentations
- Public spreadsheets
- Cached web pages

No interaction with restricted systems or private content was performed.

## Ethical Considerations

This investigation followed responsible OSINT practices by:

- Using only publicly indexed information
- Avoiding authentication pages
- Not accessing restricted resources
- Not performing vulnerability testing
- Not downloading sensitive or confidential information
