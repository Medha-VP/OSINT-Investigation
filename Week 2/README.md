# Week 2 – Search Intelligence

## Overview

Week 2 focuses on advanced search intelligence techniques used in Open-Source Intelligence (OSINT). The practical exercises build upon the foundations established in Week 1 by introducing advanced Google Dorking, the Google Hacking Database (GHDB), search automation, advanced search techniques, documentation practices, and a comprehensive Fortune 500 company investigation.

## Learning Objectives

- Apply advanced Google Dorking techniques.
- Explore the Google Hacking Database (GHDB).
- Automate repetitive search tasks.
- Compare search results from multiple search engines.
- Organize search techniques into reusable documentation.
- Conduct a structured OSINT investigation of a Fortune 500 company.
