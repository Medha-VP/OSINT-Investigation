#  Search Intelligence: Google Dork Competition & Fortune 500 OSINT Investigation

## Overview

This repository contains two practical exercises completed as part of the **Week 2 Search Intelligence** module of an Open-Source Intelligence (OSINT) learning program.

The first exercise demonstrates the use of **Google Dorking techniques** to efficiently locate publicly available information related to IBM. The second exercise presents a **Fortune 500 OSINT investigation**, applying search intelligence techniques to analyze IBM's public digital footprint using multiple search engines and advanced Google Dorks.

Both investigations were conducted using only publicly available information while following ethical and legal OSINT practices.

---

## Objectives

### Google Dork Competition

- Apply advanced Google search operators.
- Discover publicly indexed documents and resources.
- Practice efficient passive information gathering.
- Follow ethical OSINT investigation techniques.

### Fortune 500 OSINT Investigation

- Investigate IBM using publicly available information.
- Compare search results across multiple search engines.
- Use Google Dorks and GHDB queries.
- Analyze publicly available technical resources and documentation.
- Demonstrate responsible OSINT methodologies.

## Ethical Considerations

All investigations were performed using publicly available information only.

No attempts were made to:

- Access restricted systems
- Log in to authentication portals
- Exploit vulnerabilities
- Download confidential information
- Perform vulnerability assessments

The investigations strictly followed passive OSINT methodologies and ethical cybersecurity practices.
