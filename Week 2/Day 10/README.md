# Advanced Search Techniques for OSINT Investigation

## Overview

This repository contains a comparative analysis of multiple search engines used in an Open-Source Intelligence (OSINT) investigation. The project evaluates how **Google**, **Bing**, and **DuckDuckGo** index and present publicly available information about **IBM**, demonstrating the importance of using multiple search platforms during an OSINT investigation.

## Objective

- Compare search results from multiple search engines.
- Investigate IBM using publicly available information.
- Analyze differences in search indexing and result ranking.
- Understand how different search engines contribute to OSINT investigations.
- Learn the benefits of combining multiple information sources.

## Investigation Methodology

The investigation involved performing identical search queries across multiple search engines to compare the publicly available information returned by each platform.

The analysis focused on identifying:

- Official website pages
- PDF documents
- Login portals
- Technical documentation
- Research papers
- Career pages
- Security-related resources

The findings were compared to understand the strengths and limitations of each search engine.

## Key Findings

- Google provided the most comprehensive collection of official IBM resources.
- Bing revealed additional indexed pages and alternative search rankings.
- DuckDuckGo returned similar public information while offering different result ordering.
- Using multiple search engines improved the completeness and accuracy of the OSINT investigation.
- Different indexing methods can reveal unique publicly available resources.
