# Google Dork Playbook for OSINT Investigations

## Overview

This repository contains a **Google Dork Playbook** developed as part of an Open-Source Intelligence (OSINT) learning program. The playbook consolidates Google Dorks, advanced search techniques, Google Hacking Database (GHDB) queries, search automation concepts, and multi-search engine investigation methods into a single reference for efficient and ethical OSINT investigations.

## Objective

- Create a reusable reference of Google Dorks for OSINT investigations.
- Organize search techniques by their use cases.
- Consolidate concepts learned from previous practical sessions.
- Improve the efficiency of public information gathering.
- Promote ethical and responsible use of advanced search techniques.

## Investigation Methodology

The playbook was developed by consolidating Google Dorks and search techniques explored during previous practical exercises. Each technique was categorized based on its primary use case to provide a structured reference for future OSINT investigations.

The playbook emphasizes:

- Passive information gathering
- Publicly available information
- Ethical search practices
- Efficient search strategies
- Multi-source verification

## Best Practices

The playbook encourages responsible OSINT practices by:

- Searching only publicly available information.
- Respecting ethical and legal boundaries.
- Avoiding unauthorized access to systems.
- Verifying findings across multiple search engines.
- Maintaining proper documentation of investigation results.
