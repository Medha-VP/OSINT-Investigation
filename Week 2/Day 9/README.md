# Day 9 - Search Automation

## Objective
To automate repetitive Google Dork searches using Python.

## Files Included
- search_automation.py
- dorks.txt
- search_results.txt

## How It Works
1. Reads Google Dorks from dorks.txt.
2. Opens each search in the web browser.
3. Logs the search time and URL.
4. Saves all activity to search_results.txt.

## Target
IBM (ibm.com)

## Language Used
Python 3
