# Understanding the Internet – Practical Investigation Report

## Overview

This repository contains a practical investigation of **Wikipedia (wikipedia.org)** to understand the fundamental concepts of the Internet. The study explores domain names, DNS, URL structure, website architecture, security mechanisms, and the relevance of these concepts in Open-Source Intelligence (OSINT).

## Objective

- Understand how websites are identified using domain names.
- Analyze DNS records and WHOIS information.
- Learn the structure and components of URLs.
- Study website architecture and security.
- Explore the relevance of Internet fundamentals in OSINT investigations.

## Target Website

**Wikipedia (wikipedia.org)**

The investigation examines Wikipedia as a real-world example of a secure, scalable, and globally accessible website.

## Methodology

The investigation was conducted using publicly available information from trusted online resources, including:

- Official Wikipedia website
- WHOIS lookup services
- DNS analysis tools
- Network and infrastructure documentation

The collected information was analyzed to understand how Internet technologies support website accessibility, security, and performance.

## Learning Outcomes

This investigation helped develop an understanding of:

- Domain names and DNS resolution
- WHOIS records and domain registration
- URL structure and navigation
- Website architecture and organization
- HTTPS and secure communication
- Load balancing and content delivery
- Surface Web, Deep Web, and Dark Web concepts
- Application of Internet fundamentals in OSINT investigations
