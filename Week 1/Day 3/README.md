# Google Dork Collection – 50 Major Google Dorks

## Overview

This repository contains a collection of **50 commonly used Google Dorks** created for educational purposes as part of an Open-Source Intelligence (OSINT) learning program. The collection demonstrates how advanced Google search operators can be used to locate publicly available documents, research papers, government publications, datasets, presentations, and other indexed resources.

## Objective

- Learn advanced Google search operators.
- Understand the use of Google Dorks in OSINT investigations.
- Practice locating publicly available information efficiently.
- Organize commonly used search queries into a reusable reference.

## Categories Covered

The collection includes Google Dorks for:

- PDF Documents
- Excel Spreadsheets
- Academic Research
- Government Publications
- GitHub Repositories
- LinkedIn Searches
- Directory Listings
- Login and Admin Pages
- Word Documents
- PowerPoint Presentations
- Technical Reports

## Google Search Operators Used

The investigation demonstrates the use of several Google search operators, including:

- `site:`
- `filetype:`
- `intitle:`
- `inurl:`
- `intext:`

These operators help narrow search results and improve the efficiency of finding publicly available information.

## Learning Outcomes

Through this exercise, the following concepts were learned:

## Ethical Considerations

This project is intended **only for educational and defensive security purposes**.

All Google Dorks included in this repository are designed to search for **publicly accessible information**. No attempts were made to access restricted, confidential, or unauthorized systems.
