# OSINT Company Investigation – IBM

## Overview

This repository contains an Open-Source Intelligence (OSINT) investigation of **IBM (International Business Machines Corporation)** using publicly available information. The project demonstrates how OSINT techniques can be used to gather, organize, and analyze information about an organization while following ethical and legal guidelines.

## Objective

- Investigate IBM using publicly available sources.
- Analyze the company's digital footprint and online presence.
- Perform domain and DNS analysis.
- Identify publicly accessible documents using Google Dorking.
- Study the evolution of IBM's website using the Wayback Machine.
- Apply ethical OSINT methodologies for information gathering.

## Investigation Methodology

The investigation was conducted using publicly available information from trusted sources, including:

- Official IBM Website
- Google Search
- WHOIS Lookup
- DNS Lookup Tools
- MXToolbox
- DNSChecker
- Wayback Machine
- Public Company Reports

The collected information was analyzed to understand IBM's digital presence, infrastructure, publicly available resources, and online footprint.
