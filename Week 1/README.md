# Week 1 – Foundations of Intelligence

## Overview

Week 1 introduces the core concepts of Open-Source Intelligence (OSINT) and provides a foundation for intelligence gathering using publicly available information. The practical exercises focus on Internet fundamentals, search techniques, digital footprints, website analysis, and basic organizational investigations while emphasizing ethical and legal OSINT practices.

## Learning Objectives

- Understand the fundamentals of Open-Source Intelligence (OSINT).
- Learn the Intelligence Cycle and different types of intelligence.
- Explore Internet architecture and web technologies.
- Perform DNS and URL analysis.
- Use Google search operators for information discovery.
- Investigate digital footprints of organizations.
- Analyze historical versions of websites.
- Conduct a basic OSINT investigation using publicly available information.
