# Internet Archives Report – Historical Website Analysis

## Overview

This repository contains a historical website analysis conducted using the **Wayback Machine**. The project examines how websites evolve over time by comparing archived versions with their current versions. Three organizations—**Loop Health**, **Airbnb**, and **Spotify**—were analyzed to study changes in website design, branding, navigation, functionality, and services.

## Objective

- Understand how websites evolve over time.
- Learn to use the Wayback Machine for historical website analysis.
- Compare archived and current versions of websites.
- Identify changes in branding, user experience, and functionality.
- Explore the importance of Internet archives in OSINT and digital investigations.

## Investigation Methodology

The investigation involved comparing archived versions of selected websites with their current versions using the Wayback Machine. Each website was analyzed for changes in:

- Website design
- Branding
- Navigation
- Services offered
- User experience
- Interactive features
- Mobile responsiveness

The findings were documented to understand how organizations improve their digital presence over time.

## Importance of Internet Archives

Internet archives play a significant role in digital investigations by:

- Preserving historical website content
- Supporting OSINT and cybersecurity investigations
- Assisting journalists and researchers in fact-checking
- Recovering deleted or modified web content
- Documenting the evolution of organizations and technologies
